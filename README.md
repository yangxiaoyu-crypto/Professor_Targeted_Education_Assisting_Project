# 教学智能助手 Teaching AI Assistant

一个专为大学教授设计的AI教学辅助系统，帮助教师完成课程设计、教学大纲撰写和教学活动规划。

## 📚 项目简介

本项目基于丰富的教学理论文献和教学资源库，利用AI大模型技术，为大学教师提供智能化的教学设计支持。系统覆盖教学设计的核心环节，帮助教师提升教学质量。

### 核心功能

1. **课程大纲智能生成**
   - 基于泰勒原理、逆向设计、布卢姆分类等理论
   - 支持多种教学模式（BOPPPS、PBL、翻转课堂等）
   - 生成规范、专业的课程大纲

2. **参与式学习方法**
   - BOPPPS教学模式
   - 项目化学习（PBL）
   - 翻转课堂
   - 案例教学法
   - 同伴教学法
   - 工作坊模式

3. **学习目标撰写**
   - 基于布卢姆教育目标分类学
   - 基于加涅学习结果分类
   - SMART原则指导
   - 生成可测量、可评价的学习目标

4. **学生评估方案**
   - 形成性评价设计
   - 总结性评价设计
   - 评价量表（Rubric）生成
   - 公平性保障机制

5. **课程思政融入**
   - 思政元素自然融入专业课程
   - 避免"两张皮"现象
   - 润物细无声的价值引领

6. **课程方案对比优化**
   - 多维度专业评价
   - 优劣势分析
   - 改进建议生成

## 🚀 快速开始

### 前置要求

- Node.js 16+
- npm 或 yarn

### 安装步骤

1. 克隆项目
```bash
git clone <repository-url>
cd public
```

2. 安装依赖
```bash
npm install
```

3. 配置环境变量（可选）
编辑 `src/config/env.js` 文件，配置API地址

4. 启动开发服务器
```bash
npm run dev
```

5. 访问应用
打开浏览器访问 `http://localhost:3000`

### 生产构建

```bash
npm run build
```

构建产物将生成在 `dist` 目录下。

## 📋 使用指南

### 新手教师

1. 从"学习目标撰写"开始，明确课程的教学目标
2. 使用"课程大纲生成"功能创建完整的教学计划
3. 在"参与式学习"模块探索创新教学方法
4. 设计科学的"学生评估"方案

### 经验教师

1. 输入现有的课程设计
2. 使用"课程优化比较"功能获取改进建议
3. 参考知识库中的理论和案例
4. 提升教学设计的规范性和创新性

### 教学创新

1. 探索"参与式学习"模块，了解多种创新教学方法
2. 结合"课程思政"，实现价值引领
3. 使用AI辅助设计个性化教学方案

## 📚 理论基础

本系统基于以下教学理论：

1. **泰勒原理** - 目标、内容、方法、评价的系统设计
2. **逆向设计（UbD）** - 从学习结果出发设计课程
3. **布卢姆分类学** - 认知、情感、动作技能目标分类
4. **加涅九大教学事件** - 有效教学的基本流程
5. **建构主义学习理论** - 学习者中心的教学理念
6. **形成性评价与总结性评价** - 科学的评价体系
